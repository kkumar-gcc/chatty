# chatty
## How to run code 
```bash
# fork repository using
git clone https://github.com/{{username]}/chatty.git

#go to chatty folder
cd chatty

#install npm 
npm install

#run app
npm run dev

```
## Technologies
* Node.js
* MongoDB with Mongoose
* TypeScript
* Express.js & Express.js middleware
* Zod validation
