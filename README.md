# chatty
## How to run code
```bash
# fork repository using
git clone https://github.com/{{username]}/chatty.git

#go to chatty folder
cd chatty

#install npm 
npm install

#copy .env.example file in .env
cp .env.example .env

#run app
npm run dev

```
